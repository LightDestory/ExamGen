ExamGen

ExamGen is a project developed by Alessio Tudisco for the Web Programming course of the 2020/2021 academic year at the University of Catania. The project consists of two independent parts: a backend that provides the API of the service and a frontend that provides the UI you are using right now.

The backend is developed using modern and flexible technologies: NodeJS, a JavaScript runtime, MongoDB, a non-relational database based on the concept of documents, Express, a powerful framework for creating REST APIs and much more (mongoose, pdfkit, ect...).

For more technical information, you can visit the GitHub repository.

The frontend is developed using a modern, intricate, ambiguous, complex (I am crying), framework called Angular. I was on the fence about the choice, but a colleague threw me into this nightmare.... Now the boat has sailed and there's no turning back. About Angular there is not much to say because it is very technical, I can add that the UI is based on the concept of "Material Design" offered by the official library.

For more technical information, you can visit the GitHub repository.
